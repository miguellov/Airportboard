⚠️ Falta el servidor online

Despliega server.js en Render (gratis), pega la URL en api-config.js del repo y vuelve a abrir. O entra directo a https://tu-app.onrender.com/panel.html.

Recomendado: servidor en Render + URL en api-config.js → funciona en PC, celular y GitHub Pages sin configurar cada dispositivo.

La TV debe tocarse una vez para activar la voz. Si la API cambia la hora, los avisos se recalculan solos.
